## 使用方法

**Note:** 选中文本 按下快捷键
* MAC   `Cmd+.`
* WINDOWS `Ctrl+.`

### 为什么使用它

* 直接将选中的中文替换为英文
* 没有语意的中文替换为中文拼音
* 小驼峰命名
